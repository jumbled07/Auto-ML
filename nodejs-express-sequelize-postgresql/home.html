Nereis-ML is a web REST-API for training, optimizing and deploying machine learning models. It can be used to easily publish novel ML models for further use in open source

Key Features

  • It supports multiple ML frameworks, like tensorflow, keras, pytorch and others integrated through Optuna.
  • Provides Cloud Storage and Deployement, with Docker to upload multiple datasets and network files
  • Can perform efficient Hyperparameter Optimization using software framework Optuna.
  • Customers can test published models from inventory
